##  Featured Projects

<details>
<summary><b>🌿 PlantScan AI — Plant Disease Detection</b></summary>
<br/>

A deep-learning powered web application that identifies plant diseases from a leaf photo and recommends treatment, using transfer learning on three CNN backbones — **EfficientNetB0**, **MobileNetV2**, and **ResNet50** — served through a Flask backend with a custom "deep forest" themed UI, live camera capture, and quality-gated auto-capture.

| Aspect | Detail |
|---|---|
| **Stack** | TensorFlow / Keras, scikit-learn, OpenCV, Flask, HTML/CSS/JavaScript |
| **Scale** | 38-class classification (PlantVillage dataset), 3 trained CNN backbones |
| **Performance** | EfficientNetB0 — 98% accuracy · MobileNetV2 — 95% accuracy |
| **Security** | Client-side image validation, camera access restricted to HTTPS/localhost contexts |
| **Impact** | End-to-end usable diagnosis tool — photo in, treatment recommendation out |
| **Repository** | [plant_disease_detection](https://github.com/Imran-Ghafoor594/plant_disease_detection) |

This project goes beyond a typical training notebook — it includes a full two-phase transfer-learning recipe (frozen backbone → fine-tuned last layers), a live camera capture mode using `getUserMedia()`, and a purely in-browser auto-capture system that checks brightness, sharpness, and motion before firing the shutter, similar in spirit to how facial-recognition unlock systems reject poor-quality frames.

</details>
